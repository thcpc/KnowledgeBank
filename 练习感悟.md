# 腹式呼吸
腹式呼吸的作用，在运动中呼吸时也能稳定核心
练习步骤
1. 气沉丹田，就像咳嗽，目的是稳住核心
2. 稳住背部鼓胀
3. 吸气鼓上腹
4. 吐气压缩上腹
5. 3，4保持气沉丹田
6. 呼吸尽量打开腹腔，后背

学会腹式呼吸有如下好处
1.增加肺活量
2.在发力时，能稳定核心，同时能稳定五脏
3.练习的时候，通过腹腔挤压内脏，也能锻炼5脏

在换气的想象气是从喉过，这样就能保证呼吸时核心不散。
八极擤气就是这个原理。
发出力量越大，稳定要求越高，这个时候吐气时机和爆发量也就越高
